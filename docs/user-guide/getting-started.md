# Kom igång med FlickSlides

FlickSlides låter dig styra presentationer på din Mac med gester från din Apple Watch.

## Systemkrav

- **Apple Watch:** watchOS 11.0 eller senare
- **iPhone:** iOS 18.0 eller senare
- **Mac:** macOS 15.0 eller senare
- Alla enheter måste vara inloggade med samma Apple ID

## Installation

### 1. Installera på Mac

1. Ladda ner FlickSlides för Mac
2. Flytta appen till Applications-mappen
3. Starta appen – den visas i menyraden

### 2. Bevilja Accessibility-behörighet

FlickSlides behöver Accessibility-behörighet för att simulera tangentbordstryck:

1. Öppna **Systeminställningar > Integritet och säkerhet > Tillgänglighet**
2. Klicka på låset och ange ditt lösenord
3. Aktivera **FlickSlides** i listan

### 3. Installera på iPhone

1. Ladda ner FlickSlides från App Store
2. Starta appen
3. Följ onboarding-guiden

### 4. Installera på Apple Watch

Watch-appen installeras automatiskt från iPhone. Om den inte dyker upp:
1. Öppna Watch-appen på iPhone
2. Scrolla ner till FlickSlides
3. Tryck **Installera**

## Första anslutningen

### Anslut iPhone till Mac

1. Starta FlickSlides på Mac (ikon visas i menyraden)
2. Starta FlickSlides på iPhone
3. Din Mac visas under "Välj Mac" – tryck för att ansluta
4. **Första gången:** Mac frågar om du vill tillåta anslutningen
   - Välj "Tillåt och kom ihåg" för att slippa godkänna igen

### Verifiera anslutning

- **iPhone:** Grön prick och "Ansluten" visas under Mac
- **Mac:** Menyradsikonen blir ifylld, visar "Ansluten: [iPhone-namn]"

## Starta en presentation

1. Öppna din presentation i Keynote, PowerPoint, eller annan app
2. Starta presentationsläge
3. På Apple Watch: Tryck **Starta** i FlickSlides
4. Gör gester för att byta slide!
